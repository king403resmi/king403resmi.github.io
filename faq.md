# Pertanyaan Umum (FAQ)

Jawaban untuk pertanyaan yang sering diajukan calon klien, mitra, dan kandidat karir.

---

## Tentang Perusahaan

### Apa itu KING403?
KING403 adalah perusahaan teknologi digital Indonesia yang menyediakan pengembangan platform, konsultasi brand, infrastruktur cloud, dan layanan operasional support sejak 2020.

### Di mana lokasi kantor?
Headquarters operasional di **Jakarta, Indonesia**. Model kerja **remote-first** dengan optional co-working/meeting space di Jakarta.

### Berapa ukuran tim?
40+ profesional across engineering, design, DevOps, dan operations (per Q2 2026).

---

## Layanan & Proyek

### Layanan apa saja yang ditawarkan?
Enam lini utama: Platform Development, Brand Strategy, DevOps/Cloud, Customer Support, Security/Compliance, dan Data Analytics. Detail: [Layanan](layanan.html).

### Berapa lama waktu development typical MVP?
**8–12 minggu** untuk MVP platform standar. Proyek enterprise multi-modul: **3–6 bulan** dengan delivery bertahap.

### Apakah ada konsultasi gratis?
Ya — **discovery call gratis** 30–60 menit untuk memahami kebutuhan awal. Proposal formal disusun setelah scope preliminary clear.

### Apakah source code jadi milik klien?
**Ya**, untuk proyek fixed-scope — source code dan IP deliverable diserahkan ke klien sesuai kontrak (kecuali komponen open source third-party).

### Apakah KING403 handle maintenance pasca launch?
Ya — model **retainer bulanan** untuk bug fix, security patch, minor feature, dan monitoring.

---

## Teknis

### Tech stack apa yang dipakai?
React/Next.js, Node.js/Go, PostgreSQL, Redis, Docker, cloud (AWS/GCP). Detail: [Teknologi](teknologi.html).

### Apakah support mobile app?
Ya — React Native untuk cross-platform, atau PWA untuk use case lebih ringan.

### Bagaimana dengan keamanan data?
Security by design: encryption, access control, audit log, dependency scanning. Enterprise clients dapat request penetration test.

---

## Harga & Kontrak

### Bagaimana model pricing?
**Fixed price** untuk scope jelas, atau **Time & Material** untuk exploratory/R&D phase. Retainer untuk ongoing support.

### Apakah ada rate card publik?
Tidak — setiap proyek di-quote custom berdasarkan scope, timeline, dan SLA.

### Metode pembayaran?
Transfer bank (IDR). Termin typical: 30% kickoff, 40% milestone, 30% pre-launch (negotiable).
