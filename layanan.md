# Layanan KING403

Kami menawarkan **6 lini layanan** yang dapat diambil modular atau sebagai paket end-to-end. Setiap engagement dimulai dengan **discovery call gratis** dan dokumen scope (SOW) sebelum development.

---

## Ringkasan Paket

| Paket | Cocok untuk | Durasi tipikal |
|-------|-------------|----------------|
| **Starter** | Website korporat, landing page | 4–6 minggu |
| **Growth** | Platform MVP, dashboard admin | 8–12 minggu |
| **Enterprise** | Multi-modul, integrasi legacy, SLA | 3–6 bulan |
| **Retainer** | Maintenance, support, iterasi | Bulanan |

---

## 1. Pengembangan Platform Digital

### Scope
- Website & portal korporat
- Web application (SPA / SSR)
- Mobile app (React Native / PWA)
- Admin dashboard & CMS custom
- REST / GraphQL API
- Integrasi payment gateway, SMS, email, CRM

### Deliverables
- Source code + dokumentasi teknis
- Environment staging & production
- CI/CD pipeline
- Handover & training tim klien

### Tech highlights
- Arsitektur modular & testable
- Mobile-first responsive UI
- Performance budget (LCP, TTI)
- Accessibility baseline (WCAG 2.1 AA target)

**Estimasi:** mulai dari 8 minggu (MVP)

---

## 2. Konsultasi Brand & Digital Strategy

### Scope
- Brand audit & competitor analysis
- Positioning & messaging framework
- Visual identity (logo, color, typography)
- Design system & component library
- Content strategy & editorial calendar
- Social media playbook

### Deliverables
- Brand guideline PDF (30–60 halaman)
- Figma design system
- Template konten (presentasi, sosmed, email)
- Workshop brand internal (1–2 hari)

**Estimasi:** 4–8 minggu

---

## 3. Infrastruktur & DevOps

### Scope
- Cloud architecture (AWS / GCP / Azure / hybrid)
- Container orchestration (Docker, K8s)
- Infrastructure as Code (Terraform)
- CI/CD (GitHub Actions, GitLab CI)
- Monitoring (Prometheus, Grafana, alerting)
- Log aggregation & tracing
- Cost optimization review

### SLA opsi
| Tier | Uptime target | Response |
|------|---------------|----------|
| Standard | 99,5% | 4 jam |
| Business | 99,9% | 1 jam |
| Enterprise | 99,95% | 15 menit |

**Estimasi:** 2–4 minggu setup awal + retainer opsional

---

## 4. Customer Support & Operations

### Channel
- Live chat (integrasi website/app)
- Email & ticketing system
- WhatsApp Business API (via mitra resmi)
- Phone support (tier enterprise)

### Layanan
- Tier-1 / tier-2 support
- Knowledge base authoring
- SLA reporting bulanan
- QA conversation sampling
- Escalation matrix & runbook

### KPI yang dilacak
- First response time (FRT)
- Resolution time
- CSAT / NPS
- Ticket volume trend

**Model:** per-agent/month atau fixed fee per volume

## 5. Keamanan & Compliance

### Assessment
- Architecture security review
- OWASP Top 10 vulnerability scan
- Dependency audit (SCA)
- Access control & secrets management review

### Implementasi
- WAF & DDoS mitigation
- Encryption at rest & in transit
- RBAC / ABAC policy
- Audit logging
- Incident response playbook

### Compliance support
- GDPR-ready data handling
- Kebijakan privasi & retention
- DPIA assistance (Data Protection Impact Assessment)

**Estimasi:** 2–6 minggu tergantung scope audit

## 6. Data & Analytics

### Scope
- Event tracking design
- Dashboard BI (Metabase, custom)
- Funnel & cohort analysis
- A/B testing framework
- Automated reporting (PDF/email scheduled)

### Output
- Single source of truth dashboard
- Documented metric definitions
- Training interpretasi data untuk tim bisnis

## Proses Engagement Detail

### Fase 1 — Discovery (1–2 minggu)
- Stakeholder interviews
- Requirement gathering
- Technical feasibility study
- Proposal + timeline + budget

### Fase 2 — Design (2–4 minggu)
- Wireframe & user flow
- UI mockup (Figma)
- Architecture diagram
- Sign-off design

### Fase 3 — Development (iteratif, 2-week sprint)
- Sprint planning & daily standup
- Code review wajib
- Demo akhir sprint ke klien

### Fase 4 — QA & UAT
- Automated test suite
- Manual exploratory testing
- User acceptance testing dengan tim klien
- Bug bash & fix sprint

### Fase 5 — Launch
- Deployment checklist
- Rollback plan
- War room 48 jam post-launch
- Hypercare period (2–4 minggu)

### Fase 6 — Support & Iterate
- Retainer maintenance
- Feature backlog prioritization
- Performance review quarterly

---

## Model Harga

Kami tidak publikasi rate card fixed — setiap proyek unik. Faktor penentu:

- Kompleksitas fitur & integrasi
- Jumlah platform (web + mobile)
- SLA support yang diminta
- Durasi engagement

**Transparansi:** breakdown biaya per fase disertakan di proposal. Tidak ada hidden fee.
